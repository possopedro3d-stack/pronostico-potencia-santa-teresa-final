# Descripción de los datos

El estudio utiliza 5057 registros históricos de potencia efectiva de las unidades G1 y G2 de la central Santa Teresa.

Los registros presentan una frecuencia temporal de 15 minutos y corresponden al período comprendido entre el 1 de diciembre de 2024 y el 31 de marzo de 2025.

Después del preprocesamiento y de la construcción de las variables históricas, se conforma una matriz supervisada común de 5007 observaciones útiles.

La base original no se publica en este repositorio debido a consideraciones de confidencialidad y uso institucional.
